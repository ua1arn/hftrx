# Меню настроек трансивера

## Вход в меню
- **Способ входа:** Удерживайте кнопку **MENU** более 0.6 секунд.

## Сброс параметров
- **Активация:** Включите питание с зажатой кнопкой **SPLIT** → на дисплее появится запрос на сброс.
- **Подтверждение:** Повторно нажмите **SPLIT**.
- **Отмена:** Выключите питание. Другие кнопки игнорируются.
